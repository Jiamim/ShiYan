数据查询
======
##单表查询
###例1 查询全体学生的学号与姓名
```sql
SELECT Sno,Sname
FROM Student;
```
###例2 查询全体学生的详细记录
```sql
SELECT *
FROM Student
```
###例3 查询全体学生的姓名及其出生年份
```sql
SELECT Sname,2015 - Sage
FROM Student
```
>编写本文的时间是2015年。

###例4 查询全体学生的姓名、出生年份和所在院系（用小写字母表示系名）
```sql
SELECT Sname,'Year of Birth:',2015 - Sage,LOWER(Sdept)
FROM Student;
```
