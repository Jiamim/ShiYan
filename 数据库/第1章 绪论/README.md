##数据模型
###分类
根据模型应用的目的不同，可以划分为两类：
* 第一类
  * 概念模型（*信息模型，按用户的观点来对数据和信息建模，用于数据库设计*）
* 第二类
  * 逻辑模型（*按计算机系统的观点对数据建模，用于DBMS的实现*）
    * [层次模型](#层次模型)
	* 网状模型
	* 关系模型
	* 面向对象模型
	* 对象关系模型（NOSQL）
  * 物理模型（*一般用户不用考虑*）
 
###组成要素
数据模型通常由以下三部分组成：
* 数据结构
* 数据操作
* 完整性约束
