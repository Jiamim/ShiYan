概述
=======
##发展历程
|发展|名称|典型代表
|---|---|---
|第0代|机械时代|差分机、制表机、Z-3型、MARK-I
|第1代|真空管（电子管）计算机|ENIAC、EDVAC、UNIVAC-I、IBM701
|第2代|晶体管计算机|IBM7040、IBM7070、IBM7090
|第3代|集成电路计算机|IBM360、PDP-II、NOVA1200
|第4代|大规模和超大规模集成电路计算机|使用Intel芯片的为代表的计算机
|第5代|新一代计算机（人工智能计算机）|生物计算机、光子计算机、超导计算机

##种类
1989年IEEE科学巨型机委员会提出的按运算速度分类法：  
按性能从高到低（规模一般也可看做从大到小）
* 巨型机
* 大型机
* 小型机
* 工作站
* 微型机

##基本组成
计算机的基本组成是由五大部件构成：
* 运算器
  * ALU（逻辑运算单元）
  * 寄存器组R（*用于存放运算中产生的中间结果*）
* 控制器
  * PC（程序计数器）
  * IR（指令寄存器）
  * ID（指令译码器）
  * AG（地址生成器）
  * AR（地址寄存器）
  * CP（时序部件）
  * 控制信号产生部件
* 存储器
* 输入部件
* 输出部件

##分层组织结构
计算机系统的分层组织结构

|分层|名称|组成
|----|-----|----
|第6层|用户|执行的程序
|第5层|高级语言|C++、Java等
|第4层|汇编软件|汇编语言代码
|第3层|系统软件|操作系统
|第2层|机器|指令集组织结构
|第1层|控制系统|微代码或硬布线
|第0层|数字逻辑电路|电子线路、逻辑门
