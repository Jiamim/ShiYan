总线与接口组织
=========
##互连结构
在计算机中，主要的通信包括以下几种：

||描述
|---|---|
|存储器到CPU|CPU从存储器中读指令或数据，CPU要向存储器给出**地址信号**并发出**读控制信号**
|CPU到存储器|CPU向存储器中写入数据，CPU要向存储器给出**地址信号**和**写入数据**，并发出**写控制信号**
|I/O到CPU|CPU从I/O接口中读数据，CPU要向I/O接口给出**地址信号**并发出**读控制信号**，I/O接口向CPU提供**设备状态信号**和向CPU发出**中断请求**等控制信号
|CPU到I/O|CPU向I/O接口中写入数据，CPU要向I/O接口给出**地址信号**和**写入数据**，并发出**写控制信号**，I/O接口向CPU提供**设备状态信号**和向CPU发出**中断请求**等控制信号
|I/O和存储器之间|I/O接口代替CPU控制I/O设备与存储器之间进行之间数据传输（如**DMA方式**）

###互连方式
多个部件之间的通信可以由多种互联方式：
- 环形结构
- 全互连结构
- 星形结构
- 总线结构

##总线（Bus）互连
总线具有两个重要特点：
- 共享性
- 独占性

总线包括：
- 地址总线
- 数据总线
- 控制总线
- *电源线和接地线*
