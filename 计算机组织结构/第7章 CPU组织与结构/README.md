CPU的功能和组成
=======
##功能
1. 指令控制
2. 操作控制
3. 时间控制
4. 数据加工

##CPU基本组成
现代CPU组成：
- 运算器。组成：
  - ALU（逻辑运算单元）
  - AC（累加寄存器）
  - DR（数据缓冲寄存器）
  - PSW（状态条件寄存器）
  - REG（通用寄存器组）
  - 协处理器（*负责浮点数和向量运算*）
- 控制器。组成：
  - PC（程序计数器）
  - IR（指令寄存器）
  - ID（指令译码器）
  - AG（地址生成器）
  - AR（地址寄存器）
  - CP（时序发生器）
  - 操作控制器
- Cache（*早期CPU没有*）

控制器根据操作控制器的组成及产生微操作控制信号方法的不同，分为：
- 硬布线控制器：采用组合逻辑技术或门阵列实现
- 微程序控制器：采用存储逻辑实现
