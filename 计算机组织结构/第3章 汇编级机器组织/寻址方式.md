寻址方式
==========
##指令寻址方式
指令的寻址方式分为：
- 顺序方式
- 跳跃方式

##操作数寻址方式
###1. 立即寻址
```
MOV AX,2000H
```
把立即数2000H传送到累加寄存器AX
###2. 直接寻址
```
MOV AX,[2000H]
```
源地址2000H是偏移地址（有效地址），物理地址=DS+2000H  
DS为数据段。
###3. 寄存器寻址
操作数在**通用寄存器**中，就是寄存器寻址
```
MOV AX,BX
```
可得到较高运算速度
###4. 寄存器间接寻址
```
MOV AX,[BX] 
```
寄存器中存放除此之外的有效地址
###5. 寄存器相对寻址
```
MOV AX,COUNT[SI]
MOV AX,[COUNT+SI]
```
COUNT为16位偏移量的符号地址（Intel 8086环境中）
###6. 基址变址寻址
```
MOV AX,[BX][DI]
MOV AX,[BX+DI]
```
###7. 相对基址变址寻址
```
MOV AX,MASK[BX][DI]
MOV AX,MASK[BX+DI]
```
###8. 段寻址
为了扩大寻址范围而采用的技术。
###9. 复合寻址
把两种或两种以上的基本寻址方式组合在一起的寻址方式
