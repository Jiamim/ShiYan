存储系统组织和结构
==============
##分类
计算机存储器
* 主存*（属于半导体存储器）*
  * 随机存取存储器（RAM）
  * 只读存储器（ROM）
* 辅存
  * 磁介质存储器
    * 磁盘
	* 磁带
  * 光介质存储器
    * CD-ROM
	* 读写光盘
* 高速缓存（Cache）

从辅存到主存，再到Cache：
* 容量越来越小
* 速度越来越快
* 价格越来越高

###具体分类
|分类依据|类型|
|------|-----
|在计算机中的作用|主存、辅存、高速缓存
|物理存储介质和材料|半导体存储器、磁介质存储器、光介质存储器
|读写功能|RAM、ROM
|其他|易失性、非易失性
>闪存（Flash Memory）是一种半导体存储器。U盘是闪存的一种

##指令周期
指令从存储器读出的时间+指令在总线上传输的时间+指令在CPU中执行的时间
