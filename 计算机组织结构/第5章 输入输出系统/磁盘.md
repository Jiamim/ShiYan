磁盘存储器
=========
*是一种外部存储器*  
##磁盘存储器的物理组成
- 磁盘片
- 磁盘驱动器
- 磁盘控制器

##磁盘存储器的数据组织
磁盘以**盘面|磁道|扇区**的方式来进行数据组织的。
>只有被格式化的磁盘才有扇区的概念

##磁盘容量
###非格式化磁盘
- 磁盘总容量=盘面数*每面容量
  - 面容量=磁道数*每道容量
    - 磁道数=道密度*径向有效距离（有效半径）
	- 道容量=位密度*磁道周长

>磁道数也可以用**柱面数**来表示

###格式化磁盘
- 磁盘总容量=盘面数*每面容量
  - 面容量=磁道数*每道容量
    - 道容量=扇区数*每扇区容量
	
##访问时间
访问时间=寻道时间+旋转延时+传送时间

###寻道时间
###旋转延时
等于磁盘旋转一周所需时间的一半。
###传送时间
数据传送发生的时候，就是磁盘在转动而以，与平时的转速相同。并不是磁道和磁头两点接触不动，就传送了数据了。

假设磁盘旋转速度为每秒r转，每条磁道容量为N个字节，则数据传输速率为：
```
    D=rN(B/s)
```
若要传送的数据为b字节，则传送时间：
```
    t=b/rN
```
当一次读/写的字节数相当于半条磁道上的字节数时，旋转延时和传送时间相同。  
同理当一次读/写的字节数相当于整条磁道上的字节数时，传送时间就是磁盘旋转一周所需的时间。
