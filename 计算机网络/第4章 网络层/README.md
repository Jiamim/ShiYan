##IP地址
IP地址由32位二进制位组成。分为A、B、C、D、E五类，通常用点分十进制表示。按每8位一组，分为4组，不同类的IP地址，第一个数字的取值范围是不同的：

|分类|首数取值范围|第一组|第二组|第三组|第四组
|----|:------:|----|-----|------|-----
|A类|0~127|0*******|||
|B类|128~191|10******|||
|C类|192~223|110*****|||
|D类|224~239|1110****|||
|E类|240~255|11110***|||
很容易看出每一个取值范围第一个数字的差分别为：128、64、32、16。原因很容易理解
