物理层
===========
##几个计算
###调制速率
又称符号速率、码元速率或波特率。单位是波特（Baud）。  
```
    调制速率 N=1/T（s）
```
T是码元信号的持续时间。
###数据传信速率
或称数据传输速率。
```
    传信速率 R=Nlog2(M)
```
N是波特率，M是信号的电平数
###信噪比
>    信噪比（dB） = 10*log`10`（S/N）

S为信号平均功率，N为噪声平均功率。  
比如当S/N=10时，信噪比是10dB
>S/N就是信噪比，只不过通常用分贝（dB）作为度量单位，所以一般要按公式转化。  

###香农公式<kbd>重点</kdb>
香农公式指出：**信道的极限传输速率C**是
> C = W*log`2`(1+S/N)

W为信道的带宽（单位Hz）。  
**信道的带宽或者信道中的信噪比越大，信息的极限传输速率就越高。**
##物理层基本概念
###规程
由于历史原因，物理层的协议一般用“规程”表示。
###特性
|特性|描述|
|---|---
|机械特性|指明接口所用接线器的形状和尺寸、引脚数目和排列、固定和锁定装置等等
|电气特性|指明在接口电缆的各条线上出现的电压的范围
|功能特性|指明某条线上出现的某一电平的电压表示何种意义
|过程特性|指明对于不同功能的各种可能事件的出现顺序
##数据通信基础知识
###数据通信系统的模型
三大部分：
- 源系统（发送端、发送方）
  - 源点（源站、信源）
  - 发送器
- 传输系统（传送网络）
- 目的系统（接收端、接收系统）
  - 接收器
  - 终点（目的站、信宿）

###常用术语
|术语|描述
|----|-----
|消息|**通信的目的**是传送消息。如话音、文字、图像等
|数据|**消息的实体**是数据
|信号|是数据的电气或电磁的表现。依据消息的参数的取值方式的不同分为：模拟信号、数字信号
|模拟信号|（连续信号）代表消息的参数的取值是连续的
|数字信号|（离散信号）代表消息的参数的取值是离散的
|码元|时域概念。二进制编码情况下，只有两种码元：0和1
|信道|表示向某一方向传送信息的媒体（和电路不等同）。一条通信电路上往往包含发送和接收两条信道
|基带信号|（基本频带信号）来自信源的信号
####调制
基带信号往往包含较多低频成分，甚至直流成分，这两种分量并不能被许多信道所传输。因此必须对基带进行**调制**。  
有两类调制方式：

|两类调制|描述
|----|-----
|基带调制|仅仅对基带信号的波形进行变换，使它能够与信道特性相适应。变换后仍然是**基带信号**
|带通调制|使用**载波**进行调制，把基带信号的频率范围搬移到较高的频段以便在信道中传输，调制后的信号为**带通信号**

|带通调制方法|缩写|描述|举例
|----|----|-----|----
|调幅|AM|载波的振幅随基带数字信号而变化|0或1分别对应载波输出的无和有
|调频|FM|载波的频率随基带数字信号而变化|0或1分别对应于频率*f*`1`或*f*`2`
|调相|PM|载波的初始相位随基带数字信号而变化|0或1分别对应于相位0度或180度
>上述三种为**基本**的带通调制方法。为达到更高的信息传输速率，必须采用更为复杂的多元制的振幅相位混合调指方法。
比如：**正交振幅调指（QAM）**
