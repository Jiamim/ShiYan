运输层
======
##端到端通信
　　主机之间的通信实际上是主机中应用进程之间的互相通信。*IP协议虽然能报分组送到目的主机，但IP地址仅表示主机，而不能标识主机中
的应用进程。*运输层协议负责实现应用进程之间的通信，运输层向上面的应用层提供服务，它使用进程看见的就好像是在两个运输层实体之间有一条端
到端的逻辑通信信道，因此，运输层的通信又称为**端到端通信**
##TCP与UDP
|TCP|UDP
|:---:|:---:
|基于IP传输|基于IP传输
|面向连接服务|无连接服务
|有流量控制|无流量控制
|有拥塞控制|无拥塞控制
|保证可靠交付|不保证可靠性交付
|按序交付|可能失序
##TCP套接字
```
	套接字 socket = (IP地址:端口号)
```
**每一条TCP连接唯一地被通信两端的两个端点（即两个套接字）所确定。**
###端口号
- [x] 服务端使用的端口号分为两类：
- 熟知端口号（系统端口号）：取值0~1023
- 等级端口号：1024~49151
