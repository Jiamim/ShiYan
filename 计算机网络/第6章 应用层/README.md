应用层
=======
##域名系统DNS
DNS（Domain Name System）。
###域名
>比如：mail.cctv.com
* com是顶级域名（TLD）
* cctv是二级域名
* mail是三级域名

##FTP协议
文件传输协议（File Transfer Protocol）  
服务端有两个从属进程：
* 控制进程（端口号21）
* 数据传送进程（端口号20）

>###TFTP
简单文件传送协议（Trivial File Transfer Protocol）。它使用的是UDP。

##TELNET
远程终端协议。  
熟知端口号为23。

动态主机配置协议。
